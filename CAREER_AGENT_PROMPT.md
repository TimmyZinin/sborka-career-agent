# Career Scout Agent — Универсальный промпт

> Готовый промпт + архитектура AI-агента для поиска работы.
> Подходит любому IT-специалисту: разработчику, продакту, дизайнеру, маркетологу.

---

## КАК ПОЛЬЗОВАТЬСЯ

### Вариант 1: Прямо сейчас (без кода)

Скопируй промпт ниже, вставь в ChatGPT / Claude / любую LLM, заполни свой профиль — и получи:
- Список подходящих ролей на рынке
- Конкретные площадки для поиска
- Персонализированный шаблон аутрича
- Оценку своего fit по каждой роли

### Вариант 2: Построить агента (для тех, кто хочет автоматизировать)

Используй архитектуру ниже — агент будет сканировать вакансии, оценивать fit и присылать тебе лучшие варианты в Telegram.

## ПРОМПТ ДЛЯ КАРЬЕРНОГО АНАЛИЗА

Копируй и вставляй в любую LLM. Заполни блок `[ЗАПОЛНИ]`.

```
Ты — карьерный консультант с 20-летним опытом на IT-рынке России и международном рынке.
Одновременно ты рекрутер, который понимает, как думает наниматель.
Одновременно ты коуч, который помогает людям найти свою сильную позицию.

ЗАДАЧА:
Проанализируй мой профиль и дай конкретный план поиска работы / партнёрства.

МОЙ ПРОФИЛЬ:
- Имя: [ЗАПОЛНИ]
- Текущая роль: [ЗАПОЛНИ — например: backend-разработчик, продакт-менеджер, дизайнер]
- Опыт (лет): [ЗАПОЛНИ]
- Hard skills: [ЗАПОЛНИ — языки, фреймворки, инструменты]
- Soft skills: [ЗАПОЛНИ — что умеешь кроме кода]
- Индустрии: [ЗАПОЛНИ — где работал: финтех, e-commerce, edtech и т.д.]
- Что ищу: [ЗАПОЛНИ — работу в найме / фриланс / co-founder / партнёрство]
- Формат: [ЗАПОЛНИ — удалёнка / офис / гибрид]
- Зарплатные ожидания: [ЗАПОЛНИ или "рынок"]
- Чего НЕ хочу: [ЗАПОЛНИ — что точно не подходит]

ЧТО Я ХОЧУ ПОЛУЧИТЬ (дай ВСЁ по порядку):

1. ЦЕЛЕВЫЕ РОЛИ (3-5 штук)
   Для каждой роли:
   - Название роли (на русском и английском)
   - Где искать эту роль (конкретные площадки, каналы, сайты)
   - Средняя вилка зарплат / компенсации
   - Мой fit (0-10) с объяснением
   - Что нужно докачать, чтобы fit стал 10/10

2. ПОРТРЕТ НАНИМАТЕЛЯ
   - Кто эти люди (размер компании, стадия, индустрия)
   - Что у них болит (какую проблему я решаю)
   - Что они ценят больше всего (hard skills? soft skills? опыт в нише?)
   - Красные флаги — чего избегать при поиске

3. СТРАТЕГИЯ ПОИСКА
   - Конкретные площадки с приоритетами (ТОП-5)
   - Поисковые запросы (копипаст-ready для LinkedIn, hh.ru, Telegram)
   - Сколько сообщений/откликов в день для результата
   - Timeline: через сколько ожидать первые созвоны

4. ШАБЛОН АУТРИЧА
   - Холодное сообщение для каждой целевой роли (3-4 предложения)
   - Что менять под каждую вакансию

5. QUICK WINS (5 действий на сегодня)
   - Что сделать прямо сейчас, без подготовки, за 1 час

Формат: структурированный, с заголовками, без воды.
Если чего-то не хватает в моём профиле — спроси, прежде чем отвечать.
```

## ПРОМПТ ДЛЯ АГЕНТА-СКОРИНГА (для автоматизации)

Этот промпт используется внутри агента — он оценивает каждую найденную вакансию/возможность.

```
Ты — карьерный скаут. Твоя задача — оценить возможность для кандидата.

ПРОФИЛЬ КАНДИДАТА:
{profile_yaml}

ВОЗМОЖНОСТЬ:
{opportunity_text}

Оцени по шкале 0-10:

1. RELEVANCE — Насколько роль соответствует навыкам?
2. URGENCY — Насколько срочно ищут? (выше = лучше шанс)
3. COMPENSATION — Привлекательность компенсации (деньги, equity, перспектива)
4. CULTURE_FIT — Подходит ли стиль работы?
5. GROWTH — Потенциал роста (рынок, команда, продукт)

RED_FLAGS: [список, если есть]
TOTAL_SCORE: среднее 5 критериев
VERDICT: "APPLY" (>= 7) | "MAYBE" (5-6) | "SKIP" (< 5)

Если VERDICT = "APPLY":
DRAFT_MESSAGE: персонализированное сообщение от кандидата (3-4 предложения,
упомяни конкретно, чем полезен именно этому проекту).

JSON формат.
```

## ПРОФИЛЬ КАНДИДАТА (шаблон для агента)

Сохрани как `profile.yaml` и заполни под себя:

```yaml
name: "Твоё имя"
roles:
  - "Целевая роль 1"
  - "Целевая роль 2"
  - "Целевая роль 3"

hard_skills:
  skill_1: expert       # expert / advanced / intermediate
  skill_2: advanced
  skill_3: intermediate

soft_skills:
  - "Описание soft skill 1"
  - "Описание soft skill 2"

preferences:
  compensation:
    - salary: preferred
    - equity: acceptable
  location: remote_first    # remote_first / office / hybrid
  commitment: full-time     # full-time / part-time / freelance
  industries:
    preferred: [fintech, saas, edtech]
    acceptable: [marketplace, media]
    avoid: [gambling]

deal_breakers:
  - "Что точно не подходит 1"
  - "Что точно не подходит 2"
```

```
│  vc.ru                        трекинг       │

│  Deploy: Docker                             │

```

### Пайплайн:

1. **SCRAPER** (каждые 6 часов) — парсит Telegram-каналы, LinkedIn, hh.ru, vc.ru
2. **MATCHER** — LLM оценивает каждую возможность по твоему профилю (5 критериев, порог >= 7)
3. **OUTREACH** — генерирует персонализированное сообщение, отправляет тебе в Telegram с кнопками [Написать] [Пропустить] [Позже]

## FAQ

**Q: Я не программист, могу использовать?**
A: Да! Промпт для карьерного анализа работает в любой LLM без кода. Просто скопируй, заполни профиль, вставь в ChatGPT/Claude.

**Q: А если я хочу автоматизировать, но не умею кодить?**
A: Используй n8n или Make.com — визуальные инструменты автоматизации. Или попроси Claude Code / Cursor написать код за тебя.

**Q: Какой LLM лучше использовать?**
A: Claude лучше для структурированного анализа. ChatGPT лучше для креативных текстов. Для скоринга внутри агента — Claude Sonnet (быстрый и дешёвый).

**Q: Это легально — парсить вакансии?**
A: Публичные данные (открытые TG-каналы, публичные вакансии) — да. LinkedIn scraping — серая зона, лучше использовать их API или ручной поиск.
